# Spring-boot-starter

### "Spring-boot-starter" is tutorial to learn spring-boot for biginner who start learning spring-boot recently.

# Features

- Spring boot
- Thymeleaf
- H2 Database
- JDBC API
- Spring Boot Devtools
- Spring Web
- Validation
- Maven

# Requirement

- java.version : 11
- spring-boot-parent : 2.5.3
